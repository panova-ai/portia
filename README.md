# Portia

> *Portia (Latin: gate, portal) - The gateway between health data formats*

Panova's health data interchange service for importing and exporting patient records.

## Capabilities

### Import
- C-CDA (Continuity of Care Documents)
- HL7v2 (Lab results, ADT messages)
- FHIR R4/STU3

### Export
- FHIR R5 Bundle
- FHIR R4 Bundle
- C-CDA
